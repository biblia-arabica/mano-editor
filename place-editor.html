About Place Editor

The Place Editor creates one TEI place record per session, conforming to the project's place template. Each record receives a numeric ID; the exported file {id}.xml belongs in the repository folder data/places/.

Use the lookup fields to link external authorities: GeoNames (requires a configured username), GND, and Wikidata. Selecting a GeoNames result can fill the coordinates automatically. The City and Region fields link to other place records of this project.

Upload XML restores an existing place record into the form. Preview shows the TEI-XML before download.
