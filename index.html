What is <MANO>?

A lightweight workspace for digital manuscript studies

<MANO> brings together simple browser-based tools for describing, transcribing, exploring, and sharing manuscript materials.

<MANO> is an open digital environment designed for manuscript studies, academic research, and teaching. It provides a workspace where users can create and share manuscript metadata, view and modify transcriptions, and access learning resources.

The TEI Editor facilitates the structured documentation of manuscript features while offering download options for sharing, further analysis, or archival purposes.

Collection provides a shared space to explore manuscript, person, place, and work records.

This version of MANO has been adapted for use in the Biblia Arabica Project.
