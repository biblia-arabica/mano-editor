About Metadata Editor

The Manuscript Editor creates one TEI manuscript record per session, conforming to the Biblia Arabica project template. The description is organised into the accordion sections below (identification, contents, physical description, history, reproductions, and bibliography). Each record receives a numeric ID; the exported file {id}.xml belongs in the repository folder data/manuscripts/.

All references to persons, places, and works link to other records of this project via the local authority index — there are no external (Wikidata/GeoNames/GND) lookups on this page.

Upload XML restores an existing manuscript record into the form. Preview XML shows the TEI-XML before download.

Completed records may be contributed to the project's data repository, from where they are published and indexed for the other editors to link against.
